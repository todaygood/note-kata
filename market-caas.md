

# 市场上的云原生平台

## 灵雀云ACP

云原生应用架构

在应用架构层，ACP 3.0 同时支持Service Mesh和Spring Cloud两种服务治理架构。



在后Kubernetes时代，Service Mesh是微服务治理的最佳实践。Alauda Service Mesh是基于容器和Istio的微服务治理平台，提供基于Istio的应用全生命周期管理和全面微服务治理能力。同时，由于在现有企业环境下，企业技术栈大都基于Java，灵雀云深入到企业级Java开发场景，推出了Spring Cloud for Alauda，两种架构在ACP 3.0平台上进行一致管理。



为了更好地连接和管理各种API和微服务，灵雀云推出企业级API管理平台Alauda API Management Platform（AMP），支持API全生命周期管理，同时具备API能力开放运营（API Economy）和API治理能力。



后续灵雀云还将深入开发侧提供更多工具，让用户更加便捷地使用Spring Cloud和Service Mesh，实现迭代式开发。
